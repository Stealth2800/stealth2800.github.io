--- layout: page title: Bukkit Plugin Commissions --- I currently offer a private Bukkit plugin development service for a fee. Basically, the process goes like this:
  1. You request a plugin via my plugin request form.
  2. I contact you via the contact information provided in the form and I give you a price quote.
  3. Work on the plugin begins once a price is agreed on.
  4. The plugin is finished and loaded on my test server.
  5. If the plugin requires changes, I will make them. If not, we proceed to payment.
  6. Once I verify that payment has been received, I will give you a link to a folder on my Box folder containing the plugin jar archive and documentation.

Payments

I currently only accept PayPal for payments. Payments must be sent PRIOR to me sending the completed plugin.

If a plugin's cost is $40-50 or more, I will sometimes require an initial payment to guarantee that you really want the plugin. I've completed large plugins before but wasn't paid because the clients "lost interest".

What's Included?

  • Plugin JAR archive.
  • File containing documentation on the plugin's commands, permissions, configuration, and so on.
  • Source code on request.
  • Plugin setup and/or configuration on your server (by request).
  • Assistance if you encounter configuration problems.

Licensing

Note that I will not distribute the plugin myself unless you (the client) specifically allow me to.

What You CAN Do:

  • Use the plugin on any number of your own servers.

What You CANNOT Do:

  • Resell the plugin. I won't give out your private plugin(s) to other people, so I would appreciate you doing the same for me. If you wish to resell it, let me know and I will give you a portion of the resale price for referring a customer to me.
  • Distribute the plugin freely. If you want the plugin to be freely available, let me know and I will upload it to Spigot, BukkitDev, etc.
  • Claim the plugin as your own. Development is not an easy task, so I would appreciate it if you didn't take credit that belongs to me.

Updates & Bug Fixes

  • Bug fixes are free.
  • Minor updates/changes are free. (change a line of text, add a simple configuration option, etc.)
  • Larger updates may require an extra fee, as determined by me.
